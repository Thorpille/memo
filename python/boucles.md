# Les boucles

Dans les boucles  (controle de flux):
continue
break

### Boucles for

{% highlight python %}
for i in range(1, 10)

for i in range(20, 0, -10)   pas de -10

for element in liste:
	print(element)
  
for i in range(10000):
	print("Bonjour")
{% endhighlight %}

### Boucles while

{% highlight python %}
while i < 10000:
	print("Bonjour")
	i += 1
{% endhighlight %}
